# Supply chain UI icons

Drop your SVG (and optional PNG) assets here to match the reference design.

See **`docs/ASSETS_FOR_SUPPLY_CHAIN_UI.md`** for the full list and specs.

Suggested filenames:

- `icon-supplier.svg` – package/box (Supplier)
- `icon-factory.svg` – factory building
- `icon-warehouse.svg` – warehouse with optional "24H" (WH A & B)
- `icon-dc.svg` – multi-story building (Distribution Center)
- `icon-customer.svg` – two people (Customer)
- `icon-document.svg` – document/clipboard (scenario card)
- `icon-play.svg` – play/arrow (Orders button)
- `icon-bar-chart.svg` – bar chart (Analytics)
- `icon-message.svg` – speech bubble (Reflection)
- `icon-refresh.svg` – refresh arrow (Reset)
- `icon-warning.svg` – yellow triangle (node alerts)
- `icon-check.svg` – green checkmark (Customer "Met")
- `icon-advisor.svg` – stylized person (bottom panel)
- `icon-cargo-box.svg` – optional; orange box for moving items
